Куди вставити Firebase config

  1. Створи Firebase project, додай Web app і скопіюй config.
  2. Встав config у файл:
js/config/firebase.config.js
  1. Увімкни Authentication → Google.
  2. Увімкни Firestore Database і додай правила з файлу:
firebase/firestore.rules
