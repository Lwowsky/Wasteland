# Wasteland Region Forms

Clean minimal starter version.

## What is included

- Guest mode with localStorage.
- Google login module through Firebase Auth.
- Region form builder for region 1–1200+.
- Player registration form.
- Local JSON export.
- Separate language JS files for uk/en/ru/pl/de/ja/zh/ko/vi/ar.

## Firebase

Google login is ready in code, but it needs your Firebase web app config in:

`js/config/firebase.config.js`

Until that file is filled, the site works in guest/local mode.
