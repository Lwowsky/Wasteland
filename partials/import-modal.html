Налаштування / Імпорт

Завантаж список гравців, звір колонки й підготуй сайт до розподілу по турелях.

1. Звідки взяти список гравців

Завантаж файл і налаштуй регіони/зміни для розподілу.

Регіони Захоплення регіонів Вибери регіон і увімкни потрібний тумблер для 1, 2, 3 або 4 змін.
Поточне джерело
Файл ще не завантажено.
Статус імпорту
Завантаж файл, перевір колонки й натисни «Застосувати імпорт».

2. Обов’язкові колонки

Ім’я гравця, тип військ, тір, розмір маршу, розмір групової атаки, альянс, готовність бути капітаном і доступність по змінах.

3. Додаткові колонки

Захоплення іншого регіону, резервний тип військ і будь-які додаткові поля.

4. Розпізнавання змін

Перевір значення з Excel і за потреби об’єднай зміни перед імпортом.

Правила ще не застосовані.

5. Які дані показувати в таблиці

Обери, які додаткові колонки бачити після натискання «Показати всі дані».

Додаткові тіри

Додай T1–T14 і увімкни тіри, які треба показати в лімітах турелей.

6. Скидання даних
